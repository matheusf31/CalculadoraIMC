INICIANDO PROJETO
  $ react-native init AwesomeProject (nome do projeto)
  $ react-native run-android ou $ react-native start
  cria a pasta src, cria index.js, muda o import do index.js da raiz
   
CONFIGURANDO EDITORCONFIG, ESLINT E PRETTIER
  adiciona o .editorconfig e coloca as duas ultimas linhas como TRUE
  $ yarn add eslint -D

  $ yarn eslint --init
    apaga o package-lock.json
    $ yarn

  $ yarn add prettier eslint-config-prettier eslint-plugin-prettier babel-eslint -D

CONFIGURANDO DEBBUGER PRO RN
  pesquisar e baixar reactotron
  $ yarn add reactotron-react-native
  criar em src:
    pasta config:
      Reactotron.js
    
    colocar no arquivo:
    
    import Reactotron from 'reactotron-react-native';
    if (__DEV__) {
      const tron = Reactotron.configure()
        .useReactNative()
        .connect();
        
      console.tron = tron;
      
      tron.clear();
    }
    
    eslintrc: 
    global: [ ...
      __DEV__: 'readonly'
    ]
